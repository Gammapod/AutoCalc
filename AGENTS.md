# AutoCalc Agent Instructions

These instructions apply to all agent sessions in this repository.

## Required Startup Protocol

Before touching any files (including edits, deletes, or code generation), every new agent instance must:

1. Read `README.md`.
2. Read `docs/code-map.md`.
3. Use those documents to identify canonical paths, contracts, and workflows relevant to the requested task.

If either file is missing or unclear, stop and report that before proceeding with file changes.
