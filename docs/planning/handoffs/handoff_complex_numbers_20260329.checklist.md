# Fresh-Context Handoff Checklist

## Sender Readiness

- [x] Objective is one sentence and testable.
- [x] Success criteria are explicit.
- [x] Locked decisions are complete.
- [x] File list is explicit and minimal.
- [x] Out-of-scope items are explicit.
- [x] Required output schema is present.
- [x] Delegation mode is set (`review` or `implementation`).
- [x] Spawn config will use `fork_context: false`.

## Receiver Acceptance

- [ ] Response contains required sections in exact order.
- [ ] Findings cite provided inputs.
- [ ] Risks are split into blocking/non-blocking.
- [ ] Plan is decision-complete.
- [ ] Test plan is executable.
- [ ] Open questions are minimal and concrete.
